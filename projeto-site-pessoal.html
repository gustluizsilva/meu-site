Site Pessoal com IA

Criei meu site pessoal completo sem escrever uma linha de código — apenas conversando com IA.

0 linhas de código escritas
30 min do zero ao site no ar
100% gerado via conversa com IA

O Desafio

Queria um site pessoal profissional que funcionasse como currículo online — moderno, responsivo e com minha identidade visual. O problema: não sei programar.

Em vez de contratar um desenvolvedor ou usar templates genéricos, decidi testar se a IA conseguiria criar o site inteiro apenas com minhas instruções em linguagem natural.

A Solução

Usei o Claude Code como meu "desenvolvedor IA". Descrevi o que eu queria — layout, cores, seções, textos — e a IA gerou todo o código.

O processo foi iterativo: eu pedia, revisava o resultado, ajustava os detalhes e a IA refinava. Como uma conversa com um dev, só que instantânea.

Como Funcionou

  1. Criei um briefing detalhado (CLAUDE.md) descrevendo o site que eu queria
  2. A IA gerou a estrutura HTML completa com todas as seções
  3. Os estilos CSS foram criados com tema dark, responsivo e animações
  4. JavaScript para interações, animações de scroll e formulário de contato
  5. Refinei iterativamente até chegar no resultado final

Aprendizados

  • Briefing bem estruturado é a chave — quanto mais claro, melhor o resultado
  • IA não substitui o olho crítico humano — precisa de direção e revisão
  • O processo iterativo (pedir → revisar → ajustar) funciona como um produto real
  • Product Managers têm vantagem natural: saber descrever o que quer é o core skill
